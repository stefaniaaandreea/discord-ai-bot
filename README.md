# Discord AI Bot

Acest proiect este un bot Discord realizat în Python, care folosește OpenAI API pentru a genera răspunsuri automate la mesajele trimise pe un server Discord.

## Descriere

Botul citește mesajele dintr-un canal Discord, trimite conținutul lor către OpenAI API și postează răspunsul generat înapoi în același canal.

## Funcționalități

- primește mesaje de la utilizatori pe Discord
- trimite mesajele către OpenAI API
- generează răspunsuri automate
- răspunde direct în canalul Discord

## Tehnologii folosite

- Python
- discord.py
- requests
- OpenAI API

## Fișierele proiectului

- `main.py` - fișierul principal al botului
- `requirements.txt` - librăriile necesare
- `README.md` - documentația proiectului
